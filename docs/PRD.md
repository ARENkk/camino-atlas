# Camino Atlas PRD v0.1（MVP）

## 目标
用地图展示朝圣之路（Camino de Santiago）路线组与支线变体。
用户能：选路线 → 看线条 → 看介绍 → 收藏（本地贝壳袋）。

## 语言规则
- 中文为主。
- 地名必须中英双语：中文（English）。
- 起点/终点/关键节点都用该格式。

## 必展示字段
- 距离（km）
- 推荐天数（范围，如 30–35 天）
- 难度（1–5 贝壳）

## 路线组织
- RouteGroup：路线组（如 法国之路）
- RouteVariant：变体（如 经索莫斯 / 经圣希尔）
- 变体可以只在某一段分叉，其余重合。

## 页面与流程
- 首页：地图 + 路线列表 + 详情卡片 + 收藏按钮。
- 选择路线或变体时：地图高亮并自动缩放到合适视野。
- 贝壳袋页：展示收藏的变体列表，可点回地图定位。

## 收藏
- localStorage 本地保存。
- 键名：camino_favorites_v1
- 值：variantId 数组。

## 数据
- 路线内容存放在 data/routes.json。
- geometry 先留路径占位，后续再补真实 GeoJSON 线条。
